# 余票监测系统 — 用户使用手册

## 简介

本系统用于上海文化广场演出的余票监测和自动下单。注册账号后，你可以：

- 登录上海文化广场账号
- 选择演出和票档
- 启动自动监测，发现余票自动下单
- 配置微信推送，购票成功时收到通知

---

## 1. 注册

1. 打开网站（如 `https://ticket-60.site/`）
2. 点击顶部的 **注册** 标签
3. 输入 **用户名**（自定义名称，用于区分不同用户，注册后不可更改）
4. 输入 **访问密码**（由管理员提供）
5. 点击 **注册并进入**

> 注册后你的用户名和推送配置会被永久保存，下次可直接登录。

---

## 2. 登录

1. 打开网站
2. 在 **登录** 标签页输入你的 **用户名** 和 **访问密码**
3. 点击 **登录**

如果提示"用户未注册"，请先完成注册。

---

## 3. 登录剧场账号

进入系统后，在 **账号登录** 页面：

1. 输入你的上海文化广场 **手机号** 和 **密码**
2. 点击 **获取验证码**，系统会自动识别验证码（OCR）
3. 确认验证码无误后，点击 **登录**

登录成功后，右侧日志会显示"登录成功"。

---

## 4. 选择演出

切换到 **选择演出** 标签页：

1. 点击 **加载剧目**，获取当前可购票的演出列表
2. 从下拉菜单中选择一个 **剧目**
3. 选择 **场次**（日期时间）
4. 选择 **票档**（价格区间）

选中的票档会高亮显示。

---

## 5. 启动监测

切换到 **余票监测** 标签页：

1. 设置 **查询间隔**（建议 3 秒，过快可能被限流）
2. 设置 **购票数量**（默认 1 张）
3. 点击 **开始监测**

系统会自动循环查询余票：
- 显示"暂无余票"表示暂时没有票
- 显示"发现有票"会自动尝试下单
- 下单成功后会自动停止监测

**停止监测**：点击 **停止监测** 按钮即可。

---

## 6. 配置微信推送

切换到 **微信推送** 标签页，可以在购票成功时收到微信通知。支持三种推送方式：

### WxPusher（推荐）
1. 访问 https://wxpusher.zjiecode.com/ 注册并创建应用
2. 获取 **App Token**
3. 关注你的应用，获取 **UID**
4. 填入 App Token 和 UID
5. 点击 **保存** 和 **测试推送**

### Server酱
1. 访问 https://sct.ftqq.com/ 获取 SendKey
2. 填入 SendKey
3. 点击 **保存** 和 **测试推送**

### PushPlus
1. 访问 https://www.pushplus.plus/ 获取 Token
2. 填入 Token
3. 点击 **保存** 和 **测试推送**

> 推送配置会自动保存，下次登录无需重新配置。

---

## 7. 退出登录

点击左侧边栏底部的电源图标即可退出。

---

## 常见问题

**Q: 为什么提示"用户未注册"？**
A: 你需要先注册一个用户名，然后再登录。

**Q: 忘记了用户名怎么办？**
A: 用户名是你自己设置的，如果忘记请联系管理员查询。

**Q: 推送配置关闭浏览器后会丢失吗？**
A: 不会。推送配置已自动保存到服务器，下次登录会自动恢复。

**Q: 查询间隔设多少合适？**
A: 建议 3-5 秒。间隔太短（如 1 秒）可能触发网站限流。

**Q: 多个用户同时监测同一个演出会冲突吗？**
A: 不会。每个用户有独立的监测进程和账号，互不影响。

**Q: 显示"登录已过期"怎么办？**
A: 重新执行第 3 步登录剧场账号，然后重新启动监测。
